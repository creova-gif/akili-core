## HOW TO GET EACH KEY

### INSTAGRAM (do first — takes ~45 min total)
1. Go to business.facebook.com → connect all 4 IG accounts to Facebook Pages
2. Go to developers.facebook.com → Create App → Business type
3. Add "Instagram Graph API" product
4. Go to Graph API Explorer (graph.facebook.com/v18.0)
5. Select your app → generate User Access Token
6. Check these permissions:
   instagram_basic, instagram_content_publish, instagram_manage_insights,
   pages_read_engagement, pages_show_list
7. Convert short-lived → long-lived token (60 days):
   https://graph.facebook.com/v18.0/oauth/access_token
     ?grant_type=fb_exchange_token
     &client_id=YOUR_APP_ID
     &client_secret=YOUR_APP_SECRET
     &fb_exchange_token=SHORT_TOKEN
8. Get Instagram User IDs — for each account:
   https://graph.facebook.com/v18.0/me/accounts?access_token=TOKEN
   → Get Page ID → then:
   https://graph.facebook.com/v18.0/PAGE_ID?fields=instagram_business_account&access_token=TOKEN
   → ig_user_id is your IG_USER_ID for that account

### TWITTER/X (15 min)
1. developer.twitter.com → sign in as @justin_mafie
2. Create project → Create App inside project
3. Set permissions to "Read and Write"
4. Keys and Tokens tab → generate all 5 tokens
5. Copy immediately — API Key/Secret shown once

### LINKEDIN (20 min)
1. linkedin.com/developers → Create App → name: "CREOVA Akili"
2. Company: CREOVA → Add "Share on LinkedIn" + "Marketing Developer Platform"
3. Auth tab → generate OAuth token with scopes:
   r_liteprofile, w_member_social, r_organization_social, w_organization_social
4. Get Person URN:
   curl -H "Authorization: Bearer TOKEN" https://api.linkedin.com/v2/me
   → copy "id" → LINKEDIN_PERSON_URN = urn:li:person:{id}
5. Get Company URN:
   curl -H "Authorization: Bearer TOKEN" https://api.linkedin.com/v2/organizationalEntityAcls?q=roleAssignee
   → copy organization ID → LINKEDIN_COMPANY_URN = urn:li:organization:{id}
