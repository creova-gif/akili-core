# AKILI AI — Setup Guide
## Your step-by-step instructions, Justin. Follow in order.

---

## STEP 1 — Get your Telegram bot token (5 min)
1. Open Telegram on your phone
2. Search @BotFather
3. Send: /newbot
4. Name: Akili
5. Username: AkiliCREOVA_bot
6. Copy the token BotFather gives you ✅

## STEP 2 — Get your Telegram Chat ID (2 min)
1. Search @userinfobot on Telegram
2. Start it — it tells you your Chat ID
3. Copy that number ✅

## STEP 3 — Set up Replit project (10 min)
1. Go to replit.com → New Repl
2. Choose: Python template
3. Name it: akili-ai
4. Upload ALL these files into the Repl:
   - main.py
   - agents/shield.py
   - agents/pulse.py
   - agents/reach.py
   - agents/intel.py
   - agents/amplify.py
   - agents/__init__.py
   - memory/manager.py
   - memory/__init__.py
   - requirements.txt
   - .replit
   - MEMORY.md
5. Create the folders: agents/ and memory/ inside Replit ✅

## STEP 4 — Add your secrets in Replit (3 min)
In Replit, click the lock icon (Secrets) on the left sidebar.
Add these exactly:

Key: TELEGRAM_TOKEN
Value: [paste your BotFather token]

Key: ANTHROPIC_API_KEY  
Value: [paste your Anthropic key]

Key: JUSTIN_CHAT_ID
Value: [paste your Telegram chat ID number]

DO NOT put these in any file. Secrets tab only. ✅

## STEP 5 — Run Akili (1 min)
1. Click the green Run button in Replit
2. Watch the console — you should see:
   "AKILI CORE initialized — all 5 agents loaded"
   "AKILI CORE running — Telegram bot polling"
3. Open Telegram → find your AkiliCREOVA_bot
4. Send: /start
5. You should get the AKILI ONLINE message ✅

## STEP 6 — Test each agent (10 min)
Send these messages to your bot and verify responses:

Test SHIELD: "check security status"
Test PULSE: "create a post about CREOVA Music for Instagram"
Test REACH: "draft an email to a potential client about CREOVA Solutions"
Test INTEL: "research GoPay competitors in Tanzania"
Test AMPLIFY: "create a music release campaign strategy"

## STEP 7 — Always-on (Replit Core)
To keep Akili running 24/7:
1. Upgrade to Replit Core (~$20/month)
2. This keeps your Repl always running even when you close the browser
3. Alternative: Use UptimeRobot (free) to ping your Repl every 5 min
