## 1. INSTAGRAM (Most complex — do first)

### What you need:
- A Facebook Page connected to each Instagram account
- Meta Business account (business.facebook.com)
- Meta Developer app

### Steps:
1. Go to business.facebook.com → create account if you don't have one
2. Connect all 4 Instagram accounts to Facebook Pages:
   - @jj_mafie → your personal Facebook Page
   - @creovasolutions → CREOVA Solutions Page
   - @creativeinnovation__ → CREOVA Music Page
   - @sankofastudio__ → Sankofa Studio Page
3. Go to developers.facebook.com → Create App → Business type
4. Add "Instagram Graph API" product to your app
5. Go to Graph API Explorer → generate User Access Token
6. Select permissions:
   - instagram_basic
   - instagram_content_publish
   - instagram_manage_insights
   - pages_read_engagement
   - pages_show_list
7. Click "Generate Access Token"
8. Convert to Long-Lived Token (60 days):
   GET https://graph.facebook.com/v18.0/oauth/access_token
     ?grant_type=fb_exchange_token
     &client_id={APP_ID}
     &client_secret={APP_SECRET}
     &fb_exchange_token={SHORT_TOKEN}
9. Get each account's Instagram User ID:
   GET https://graph.facebook.com/v18.0/me/accounts?access_token={TOKEN}
   → Find your Page ID → then:
   GET https://graph.facebook.com/v18.0/{PAGE_ID}?fields=instagram_business_account&access_token={TOKEN}
   → This gives you the IG_USER_ID

### Add to Replit Secrets:
IG_PAGE_ID_JJ, IG_USER_ID_JJ, IG_TOKEN_JJ (repeat for each account)

## 3. LINKEDIN (Do third)

### Steps:
1. Go to linkedin.com/developers → Create App
2. App name: "CREOVA Akili" | Company: CREOVA Solutions
3. Under Products, request:
   - "Share on LinkedIn" (for personal posts)
   - "Marketing Developer Platform" (for company page)
4. Go to Auth tab → generate OAuth 2.0 token
5. Scopes needed: r_liteprofile, w_member_social, r_organization_social, w_organization_social
6. Use OAuth 2.0 flow to get your access token
7. Get your Person URN:
   GET https://api.linkedin.com/v2/me
   → Copy the "id" field → your URN = urn:li:person:{id}
8. Get your Company URN:
   GET https://api.linkedin.com/v2/organizationalEntityAcls?q=roleAssignee
   → Copy your CREOVA Solutions organization ID
   → URN = urn:li:organization:{id}

### Add to Replit Secrets:
LINKEDIN_ACCESS_TOKEN, LINKEDIN_PERSON_URN, LINKEDIN_COMPANY_URN

### Important:
LinkedIn tokens expire every 60 days — set a reminder to refresh

## 4. GMAIL (Do fourth)

### Steps:
1. Go to console.cloud.google.com
2. Create new project → name: "CREOVA Akili"
3. Enable Gmail API:
   APIs & Services → Library → search "Gmail API" → Enable
4. Create credentials:
   APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID
   → Application type: Desktop App → name: "Akili Gmail"
5. Download the JSON file → rename to gmail_credentials.json
6. Upload gmail_credentials.json to Replit under config/ folder
7. First time running, Akili will open a browser auth window
8. Authorize with your Gmail account
9. gmail_token.json will be auto-created (saved in config/)

### Add to Replit Secrets:
GMAIL_ADDRESS → your Gmail address
GMAIL_CREDENTIALS_PATH → config/gmail_credentials.json

### Important:
First authorization needs to happen once in Replit shell:
  python integrations/gmail.py
→ Click the link it generates → authorize → done forever

## 6. GITHUB (Fastest — 2 minutes)

### Steps:

2. Personal Access Tokens → Tokens (classic) → Generate new token

   ✅ read:org (read org data)

6. Click Generate Token → copy immediately

### Add to Replit Secrets:
GITHUB_TOKEN → your generated token
